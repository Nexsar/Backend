| **ethers.js** | interact with blockchain |
| **OpenAI**    | generate contents        |

---

## Installation Guide

> Clone Repo:
```sh
git clone https://github.com/Nexsar/Backend
```

> Setup:
```sh
cd distributor
npm i
npm i -g typescript # if no typescript
```

> Run:
```sh
tsc -b
node dist/index.js
```
